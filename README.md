# Swip'n Style 👗

> Marketplace C2C de vêtements et accessoires — Suisse romande

## Stack technique

- **Frontend** : React 18 + Vite 5
- **Déploiement** : Netlify (CD automatique via GitHub)
- **Base de données** *(à venir)* : Supabase (PostgreSQL)
- **Paiements** *(à venir)* : Stripe Connect

## Lancer en local

```bash
npm install
npm run dev
```

L'app tourne sur [http://localhost:5173](http://localhost:5173)

## Déployer

Chaque `git push` sur `main` déclenche un redéploiement automatique via Netlify.

```bash
git add .
git commit -m "description de la modification"
git push
```
